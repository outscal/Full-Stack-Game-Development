**[Rules]**

- Sudoku is played on a grid of 9 x 9 spaces.
- Each of the digits 1-9 must occur exactly once in each row.
- Each of the digits 1-9 must occur exactly once in each column.
- Each of the digits 1-9 must occur exactly once in each of the 9 3x3 sub-boxes of the grid.
- Check Rules in detail - <a href="https://sudoku.com/how-to-play/sudoku-rules-for-complete-beginners/" target="_blank">here</a>

**[How will user Play in our Project]**

- In each step, the user should input anyone commands out of 3 commands given below in the same format:-
<br><br>
- 1 x y number <br>
x → row number, y → column number, number → digit (1-9) that user wants to place
- 2 <br>
undo the last step played by the user
- -1
<br>exit the game
- Add one hint option also and implement it in your own way
<br><br>
- After each step, the code should check whether the step taken by the user is valid or not, if not valid print the message and repeat the above step.
- For this project, make sure that the user is not able to undo the last step more than 5 times.

**[Task]**

- Print the necessary rules and instructions on how to play at the start of the game.
- Take user name as input.
- Create a 2D matrix of the size 9*9.
- Play the game until the user wins it or want to leave it.
    1. Take input from the user as defined above in **[How will user Play in our Project].**
    2. After taking input show the status of the matrix. Also performs the validation step and inform the user in case of an invalid step.
    3. If the player wins the game then stop the game and print the user name with a message.
    4. if the player exits the game, print some different messages.
