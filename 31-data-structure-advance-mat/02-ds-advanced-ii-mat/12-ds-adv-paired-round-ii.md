<details>
<summary>Structure of Paired Round Interviews</summary>
<p>

- **[Round 1] → 30 mins**
    1. **Person 1(P1) → Interviewer** and **Person 2(P2) → Interviewee**
        1. Time: 20 mins
        2. Ask all the questions given in the link next to this page.
    2. **Feedback discussion** (verbal, give detailed feedback to P2)
        1. Time: 10 mins.
        2. Share how he/she can improve.
- **[Round 2]**
    1. Repeat the first 30m
    2. **Person 2(P2) → Interviewer** and **Person 1(P1) → Interviewee**

- **[Open Discussion]**
    1. *20-30 mins* + research on all the questions asked. Prep time for the next round.
    2. You will be evaluated upon the interaction, so make sure you are sharing some important points here, you can consider this as a technical group discussion round.
- **[Round 3]**
    1. 30m (P1 → P2 same set of questions as in Round 1).
    2. This round is basically to test the improvement in answers so we will have exact same questions but P1's answers should have been improved.
    3. *form submission on the feedback based on improvement.*
    4. Based on how P1 fills the form, P1's grades will get decided, this will be a final submission for the day **
- **[Round 4]**
    1. 30m (P2 → P1 same set of questions as in Round 2)
    2. This round is basically to test the improvement in answers so we will have exact same questions but P1's answers should have been improved.
    3. *form submission on the feedback based on improvement.*
    4. Based on how P2 fills the form, P2's grades will get decided, this will be a final submission for the day. **

</p>
