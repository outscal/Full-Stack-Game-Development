Implement a Wave-based Enemy Spawning System



- Use a scriptable object to define wave properties like the number and types of enemies in each wave.
- Create a controller that reads from this scriptable object and spawns the specified enemies for each wave.
- Progress to the next wave only when all enemies from the current wave are destroyed.
- A UI and sound cue should be in the game when each new wave starts.
