### Q1. How current architecture can differentiate between BulletController and  EnemyController with single type T?
- C# generics help us to differentiate class types automatically
- BulletPool and EnemyPool inherit from GenericObjectPool which derives two independent lists of PooledItems. **(correct)**
- GenericObjectPool being a parent class can differentiate between two children pools.
- BulletController and EnemyController both are controllers hence controller can differentiate between the objects.
