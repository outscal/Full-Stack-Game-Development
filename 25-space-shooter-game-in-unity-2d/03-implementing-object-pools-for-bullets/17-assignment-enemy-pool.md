# Enemy Pool

### [TASKS]

- After Completing first task, on the same branch, create an **Enemy Pool** from scratch
  1. Create an enemy pool class
  2. Initialize a collection (Pooled Enemies) to hold the enemy objects.
  3. Instantiate enemy objects
  4. Implement a GetEnemy() method in the enemy pool class.
  5. Activate and position the enemies
  6. Return enemy to the pool
