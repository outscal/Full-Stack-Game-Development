## **Why did Unity Choose C#?**


---

1. **Easy Cross-Platform Support**: *C#*'s *.Net Framework* makes it easy to deploy Unity games on multiple platforms, including Windows, macOS, Linux, and various mobile platforms.
2. **Easy to learn for beginners**
3. **Balance of Performance and Simplicity**
4. 1. Although C++ provides better performance because it is closer to hardware. But C# offers a good balance between performance and simplicity
  2. To create a lot of games, the ease of C# outweighs the slight performance advantages that C++ might offer
    These games can be:
  3. 1. Games by Indie Studio:  Hollow Knight, Superhot, Broforce
    2. Games** **by Small to Mid Studio: Subnautica, Pillars of Eternity
    3. Games by Big Studios like: Kerbal Space Program, Rust, Assassin's Creed Identity
5. There are other technical reasons, too, like: **.Net Core** and **Mono framework**. You can read about and understand these reasons when you have an even deeper understanding of C#
6. **Managed Language**: C# is a managed language. Therefore, it can handle memory management and garbage collection on its own. This is a huge relief to developers because they don't have to worry about memory leaks and other memory issues. 



## **C# Beyond Unity**


---

C# comes with lots of advantages:

- **Backed by Microsoft**: C# gets very strong support, regular updates and it is used by a lot of Microsoft products and services.
- **.NET Framework and .NET Core:** C# is the primary language designed for the .NET ecosystem. This simplifies a lot of programming tasks and ensures high performance and security. Making C# a versatile choice for web, mobile and desktop applications.
- **Performance**: C# and the .NET runtime offer performance that's competitive with languages like Java and, in some cases, approaches C++ speeds, especially with recent improvements in .NET.
- **Versatility:** While some other languages specialise in specific niches, C# can be used for a wide range of web to mobile to desktop applications.



**Applications that use C#**

1. **Microsoft Office Suite**: Word, Excel, Outlook, etc.
2. **Stack Overflow** is built using C#
3. **Visual Studio** (Microsoft's Flagship IDE) is largely written in C#
4. **Spotify's** Windows application is developed using C#
5. **Azure Cloud Services**: Many of Microsoft's Azure cloud services are implemented using C#.
