# Playing with C#


---



Whenever we learn something new, we always start with the basics.

Now, who will tell me what the most basic thing is in C++??



It’s printing messages in the console….



Think about the dialogue of a movie you love the most.

For me, it’s *Avengers*!



Now, you have to print that dialogue in the console….

But how can we do it??

In C++, we used to do that using `cout`, and in C#, we can do that using the `Console.WriteLine()`.



```csharp
using System;

class Program {
    static void Main() {
        // Write your C# code here
        Console.WriteLine("Avengers Assemble!");
    }
}
```



> **💡PRO TIP:**
> Just like you write code inside `**main()**` in **C++** to execute it, similarly, you have to write code inside `**Main()**` in **C#**.
> The catch is that the `**Main()**` is inside a `**class**`.
