# Data Structures used in Game Development


---



![ ](/Full-Stack-Game-Development/images/8bfece6a5315c242.png)



1. **Inventory System**
2. - Each slot in the inventory can be represented as an element in a structure, allowing for easy access and management of items like weapons, potions, or equipment.
3. **Particle System**
4. - A particle system simulates effects like smoke, fire, or magic by managing a collection of many small particles. This system uses data structures to track each particle's properties, such as position, velocity, and lifespan, allowing for dynamic and realistic visual effects in games.
5. **Pathfinding**
6. - NPCs and AIs often determine the shortest path using data structures
7. **Creating Boards in board games**
8. - In games like LUDO, Minesweeper and Chess, boards are created using data structures
9. **Textures**
10. - Images are a collection of information about what pixel holds what colour. This information is collectively stored using a data structure
11. **Game AI → Tasks are scheduled in a sequence**
12. - An AI can have 10 tasks to do in a particular sequence. These tasks are arranged using priorities in a data structure.



These were some of the applications in Game development.

Data Structures are also used in other aspects.



# Data Structure used in daily life


---

1. **Photo Galleries**
2. - Think of a photo gallery on your phone or computer. Images are stored sequentially, allowing you to quickly access any photo by its position in the gallery.
3. **Music Playlists**
4. - Consider a music playlist on platforms like Spotify or YouTube. You can add songs to the playlist, remove them, or skip to the next one.
5. **Contacts on Your Phone**
6. - Your phone's contact list, where you can instantly find a contact's details by searching their name.



These are just the surface-level applications of data structure.
