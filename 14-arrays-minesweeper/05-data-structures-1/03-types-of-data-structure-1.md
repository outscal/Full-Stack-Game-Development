# Primitive data structures

Basic data structures that cannot be further divided are called **primitive data structures**.

Example: `int`, `char`, `float`



# Non-primitive data structures

Data structures that can be used for other complex storages are called **non-primitive data structures**. Our learnings will majorly happen over non-primitive data structures. Non-primitive data structures are further divided into** linear and non-linear data structures. **



# Linear Data Structures

A data structure is called linear if its elements are arranged in a sequence, one after the other.

![ ](https://outscal-assets.s3.ap-south-1.amazonaws.com/production/LMS/6675375f717323b2671e88fc/02_28_2025__11_33_36.png)



**Example Usage:**  Shopping list! You can add items in one after the other.



# Non-Linear Data Structures

In **non-linear data** structure elements are **not **arranged in sequence. Trees and graphs are widely used non-linear data structures.



![ ](https://outscal-assets.s3.ap-south-1.amazonaws.com/production/LMS/6675375f717323b2671e88fc/03_11_2025__06_11_51.png)



**Example Usage:** A family tree that shows the relationships within a family. Here, each family member is connected to multiple members.
