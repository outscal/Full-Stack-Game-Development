<h3>Q2. What is the time complexity of deleting and element form an array and why?</h3>
<div class="oq-opts">
  <button type="button" class="oq-opt" data-i="0">O(n) - Because deleting an element requires shifting all subsequent elements in the array to fill the gap left by the deleted element.</button>
  <button type="button" class="oq-opt" data-i="1">O(1) - Since arrays provide constant-time access to elements, deletion can also be accomplished in constant time.</button>
  <button type="button" class="oq-opt" data-i="2">O(log n) - As modern data structures employ efficient deletion algorithms such as binary search trees to optimize the process.</button>
  <button type="button" class="oq-opt" data-i="3">O(n^2) - Due to the nested iterations required to relocate elements after deletion, leading to a quadratic time complexity.</button>
