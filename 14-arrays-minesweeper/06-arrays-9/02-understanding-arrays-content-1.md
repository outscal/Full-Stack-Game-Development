## Biscuit Packet and Arrays


---

- The packet is filled with multiple biscuits
  Similarly, an array is also made up of multiple elements
- You cannot put 11 biscuits in a packet of 10!
  The packet size is fixed. Similarly, the size of arrays is also fixed
- You can directly pick up any biscuit you want, 
  Similarly, you can access any array element(biscuit) you want
- A packet of biscuits can’t hold a strawberry. A packet can only have 1 type of edible
  Similarly, an array can only hold a single type of data. 
  An Integer array can have only integers and a character array can only have characters.
- All biscuits are stored together in a sequence,
  Similarly, all elements of an array are stored contiguously one after the other in memory



So here are the key points about an array:

🔸An **array** is a data structure that stores multiple elements of the **same data type** in **contiguous memory locations**. 

🔸The **size of an array is fixed**. 

🔸Each element in the array can be accessed directly. 

🔸Arrays store data in a **sequential manner(Linear Data Structure)**.
