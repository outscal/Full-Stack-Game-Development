## Array-of-Arrays


---



![](/Full-Stack-Game-Development/images/37da263d19f582df.png)

**Array-Of-Arrays??**



In my school, during morning prayer, my teacher would say, ‘Hey you! 5th from the front, keep quiet!’

Here, **5th from the front** is my **position in a row**—this is like **index j** inside an array.

But in a **2D array**, we also need to specify **which row** we are talking about. 

So if I were in the **3rd row, 5th position**, my **i index = 3** and **j index = 5**.



![](/Full-Stack-Game-Development/images/71b6542fb8a84667.png)

**2D Array**

***i:*** Will point to an array

***j:*** Will point to an element inside that array
