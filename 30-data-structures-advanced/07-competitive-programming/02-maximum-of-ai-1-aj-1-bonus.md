# **Chapter 7 - Assignment - 2 - Maximum of (A[i] - 1) * (A[j] - 1) (Bonus)**

**[Task]**

**Problem**
- You are given an array A of n integer numbers.
- Choose two indices i & j and return the maximum of *(A[i] - 1) * (A[j] - 1)*.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the required answer.

**Constraints**
- 1 ≤ n ≤ 1000
- 0 ≤ A[i] ≤ 10^5

**Example**
- **Sample Input** <br>
    4<br>
    1 5 4 5<br>
- **Sample Output**<br>
    16<br>
