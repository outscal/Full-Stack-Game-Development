# **Chapter 7 - Assignment - 1 - Single Element In a Sorted Array (Bonus)**

**[Task]**

**Problem**
- You are given a sorted array A of n integer numbers, where every element appears exactly twice, except for one element which appears exactly once.
- Find this single element that appears only once.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the element that appears only once.

**Constraints**
- 1 ≤ n ≤ 10^5
- 0 ≤ A[i] ≤ 10^5

**Example**
- **Sample Input -** 
    5
    1 1 2 3 3
- **Sample Output -**
    2

Follow Up (Must Try) - Your solution should run in O(log n) time and O(1) space
