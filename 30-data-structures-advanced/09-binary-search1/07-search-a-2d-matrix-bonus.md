**Problem - Search a 2D Matrix**
- Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:
- Integers in each row are sorted from left to right.
- The first integer of each row is greater than the last integer of the previous row.

**Input Format**
- The first line contains two integers m, n.
- Next m lines contain n integers elements of the 2D matrix.
- The last line contains *the target,* value to be searched in the 2D matrix

**Output Format**
- Print "Yes", if *the target* is found else "No" (without quotes).

**Constraints**
- 1 ≤ M, N ≤ 1000
- 1 ≤ M + N ≤ 2000
- -10^6 <= matrix[i][j] <= 10^6

**Example 1-**
- **Sample Input** <br>
    3 4<br>
    1 3 5 7<br>
    10 11 16 20<br>
    23 30 34 60<br>
    3<br>
- **Sample Output**<br>
    Yes
