# **Chapter 9 - Assignment - 7 - Valid Perfect Square (Bonus)**

**[Task]**

**Problem - Valid Perfect Square**
- Given a positive integer *num*, write a function that returns True if num is a perfect square else False.
- Find whether *num* is a perfect square or not.
- Follow up: Do not use any built-in library function such as sqrt.

**Input Format**
- The first line contains an integer *num*.

**Output Format**
- Print "Yes" if *num* is perfect square else "No" (without quotes).

**Constraints**
- 1 ≤ num ≤ 2^31 - 1

**Example 1**
- **Sample Input** <br>
    8<br>
- **Sample Output**<br>
    No<br>

**Example 2**
- **Sample Input** <br>
    4<br>
- **Sample Output**<br>
    Yes<br>
