**Problem - Median of Two Sorted Arrays**

- Given two sorted arrays nums1 and nums2 of size M and N respectively, return the median of the two sorted arrays.
- The overall run time complexity should be O(log (M + N)).

**Input Format**
- The first line contains an integer M, denoting the number of elements of the 1st array *num1*.
- The next line contains M space-separated integers, denoting the elements of the 1st array *num1*.
- The nest line contains an integer N, denoting the number of elements of the 2nd array *num2*.
- The next line contains N space-separated integers, denoting the elements of the 2nd array *num2*.

**Output Format**
- Print the required median.

**Constraints**
- 1 ≤ M, N ≤ 1000
- 1 ≤ M + N ≤ 2000
- -10^6 <= nums1[i], nums2[i] <= 10^6

**Example 1**
- **Sample Input** <br>
    2<br>
    1 3<br>
    1<br>
    2<br>
- **Sample Output** <br>
    2.000<br>

**Example 2**

- **Sample Input** <br>
    2<br>
    1 3<br>
    2<br>
    2 4<br>
- **Sample Output**<br>
    2.5000<br>
