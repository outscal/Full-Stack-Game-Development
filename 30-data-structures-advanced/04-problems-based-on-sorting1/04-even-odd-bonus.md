# **Chapter 4 - Assignment - 1 - Even Odd (Bonus)**

**[Task]**

**Problem**
- You have been given an array *A* consisting of N integers.
- Move all the even integers at the beginning of the array followed by all the odd integers.
- Return any array that satisfies this condition.

**Input Format**
- The first line contains a single integer N denoting the size of array *A*.
- The next line contains N space-separated integers denoting the elements of array *A*.

**Output Format**
- Print the final array, in case of multiple answers print any of them.

**Constraints**
- 1 ≤ N ≤ 1000
- 1 ≤ A[i] ≤ 1000

**Example**
- **Sample Input** <br>
    5<br>
    1 2 3 4 5<br>
- **Sample Output**<br>
    2 4 1 3 5<br>
        
    Possible Other Answers - 4 2 1 3 5, 2 4 3 1 5, 2 4 1 5 3, 2 4 5 1 3, etc..
