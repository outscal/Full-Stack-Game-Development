# **Chapter 19 - Assignment - 3** Largest triplet product in a stream (Bonus)

**[ Task ]**

**Problem**
- Given a stream of integers represented as A[].
- For each index, i from 0 to n-1, print the multiplication of largest, second-largest, third-largest element of the subarray A[0…i]. If i < 2 print -1.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the required answer array.

**Constraints**
- 1 ≤ n ≤ 100000
- 0 ≤ A[i] ≤ 100000

**Example**
- **Sample Input -** <br>
    5<br>
    1 2 3 4 5<br>
- **Sample Output -**<br>
    -1 -1 6 24 60<br>
