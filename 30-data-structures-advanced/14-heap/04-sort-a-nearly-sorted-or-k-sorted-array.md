# **Chapter 19 - Assignment - 1 - Sort a nearly sorted (or K sorted) array**

**[ Task ]**

**Problem**
- You are given an array A of n integer numbers, where each element is at most k away from its target position.
- For example, let us consider k is 2, an element at index 7 in the sorted array, can be at indexes 5, 6, 7, 8, 9 in the given array.
- Devise an algorithm that sorts the given array in **O(n log k)** time.

**Input Format**
- First line: n denoting the number of array elements and k.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the sorted array.

**Constraints**
- 1 ≤ n ≤ 100000
- 1 ≤ k ≤ 10000
- 0 ≤ A[i] ≤ 100000

**Example**
- **Sample Input -** <br>
    7 3<br>
    6 5 3 2 8 10 9<br>
- **Sample Output -**<br>
    2 3 5 6 8 9 10<br>
