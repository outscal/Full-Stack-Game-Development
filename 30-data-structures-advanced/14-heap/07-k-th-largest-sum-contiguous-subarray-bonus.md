# **Chapter 19 - Assignment - 4 - K-th Largest Sum Contiguous Subarray**  (Bonus)

**[ Task ]**

**Problem**
- Given an array A of n integers.
- Write a program to find the K-th largest sum of contiguous subarray within the array of numbers that has negative and positive numbers.

**Input Format**
- First line: n denoting the number of array elements and k.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the required answer.

**Constraints**
- 1 ≤ n ≤ 1000
- 1 ≤ k ≤ 1000
- 0 ≤ A[i] ≤ 100000

**Example**
- **Sample Input -** <br>
    3 3<br>
    20 -5 -1<br>
- **Sample Output -**<br>
    14<br>
- **Explanation -** <br>
All sums of contiguous subarrays are (20, 15, 14, -5, -6, -1), so the 3rd largest sum is 14.
