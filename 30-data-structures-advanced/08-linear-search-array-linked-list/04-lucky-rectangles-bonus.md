# **Chapter 8 - Assignment - 1 - Lucky Rectangles (Bonus)**

**[Task]**

**Problem**
- You have N rectangles. A rectangle is lucky if the ratio of its sides is in between [1.6, 1.7], both inclusive.
- Your task is to find the number of lucky rectangles.

**Input Format**
- First line: Integer N denoting the number of rectangles.
- Each of the following N lines: Two integers W, H denoting the width and height of a rectangle.

**Output Format**
- Print the answer in a single line.

**Constraints**
- 1 ≤ N ≤ 10^5
- 1 ≤ W, H ≤ 10^9

**Example**
- **Sample Input**<br> 
    5<br>
    10 1<br>
    165 100<br>
    180 100<br>
    170 100<br>
    160 100<br>
- **Sample Output**<br>
    3<br>
