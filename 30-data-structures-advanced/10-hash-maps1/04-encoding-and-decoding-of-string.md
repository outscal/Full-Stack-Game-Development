# **Chapter 10 - Assignment - 1 - Encoding and Decoding of String**

**[Task]**

- TinyURL is a URL shortening service where you enter a URL such as <a href="https://leetcode.com/problems/design-tinyurl" target ="_blank">https://replit.com/problems/design-tinyurl</a> and it returns a short URL such as <a href="http://tinyurl.com/VKR1234" target ="_blank">http://tinyurl.com/VKR1234</a>.
- Design a class to encode a URL and decode a tiny URL.
- There is no restriction on how your encode/decode algorithm should work. You just need to ensure that a URL can be encoded to a tiny URL and the tiny URL can be decoded to the original URL.
- **Input Format**
    - First line: String S to be encoded and decoded.
- **Output Format**
    - Print the encoded string and decoded string.
    - Verify the output by matching decoded string to input string.
- **Constraints**
    - 1 ≤ |S| ≤ 10^5
