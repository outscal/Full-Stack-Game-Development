# **Chapter 11 - Assignment - 4 - Count Largest Group (Bonus)**

**[Task]**

**Problem**
- Given an integer n.
- Each number from 1 to n is grouped according to the sum of its digits.
- Return the number of groups having the largest size.

**Input Format**
- First line: Integer n.

**Output Format**
- Print the required answer.

**Constraints**
- 1 ≤ n ≤ 10^6

**Example - 1**
- **Sample Input**<br> 
    2<br>
- **Sample Output**<br>
    2<br>

**Example - 2**
- **Sample Input** <br>
    13<br>
- **Sample Output**<br>
    4<br>
