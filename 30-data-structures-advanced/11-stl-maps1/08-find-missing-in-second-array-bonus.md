# **Chapter 11 - Assignment - 5 - Find Missing in Second Array (Bonus)**

**[Task]**

**Problem**
- Given two arrays, find numbers that are present in the first array, but not present in the second array.

**Input Format**
- First line: Integer N denoting the size of the first array.
- Next line: N space-separated integers denoting elements of the first array.
- Next line: Integer M denoting the size of the second array.
- Next line: M space-separated integers denoting elements of the second array.

**Output Format**
- Print required numbers.

**Constraints**
- 1 ≤ N, M ≤ 10^5
- 1 ≤ arr1[i], arr2[i] ≤ 10^9

**Example - 1**
- **Sample Input** <br>
    7<br>
    3 4 7 1 2 8 9<br>
    5<br>
    1 2 3 4 7<br>
- **Sample Output**<br>
    8 9<br>

**Example - 2**
- **Sample Input** <br>
    5<br>
    1 2 33 4 5<br>
    5<br>
    1 2 3 4 5<br>
- **Sample Output**<br>
    33<br>
