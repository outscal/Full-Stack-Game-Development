# **Chapter 11 - Assignment - 2 - Minimum delete operations (Bonus)**

**[Task]**

**Problem**
- Given an array of n integers, elements may repeat.
- You can delete one element in one operation.
- Find the minimum number of operations required to make all elements equal.

**Input Format**
- First line: Integer N denoting the size of the array.
- Next line: N space-separated integers denoting elements of the array.

**Output Format**
- Print the minimum number of operations required.

**Constraints**
- 1 ≤ N ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9

**Example - 1**
- **Sample Input**<br> 
    7<br>
    3 4 7 1 2 9 8<br>
- **Sample Output**<br>
    6<br>

**Example - 2**
- **Sample Input** <br>
    7<br>
    3 4 2 1 2 9 2<br>
- **Sample Output**<br>
    4<br>
