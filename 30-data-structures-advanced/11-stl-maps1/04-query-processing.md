# **Chapter 11 - Assignment - 1 - Query Processing**

**[Task]**

**Problem**
- Given an array of N integers and an integer Q denoting the number of queries.
- In each query you will be given one integer and you have to find the frequency of that integer in the array.
- If the integer doesn't exist in the array, print frequency as 0.

**Input Format**
- First line: Integer N denoting the size of the array.
- Next line: N space-separated integers denoting elements of the array.
- Next line: Q denoting the number of queries.
- Next Q lines: One integer whose frequency is to be found.

**Output Format**
- Print Q lines denoting answers to each query.

**Constraints**
- 1 ≤ N, Q ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9

**Example**
- **Sample Input**<br> 
    7<br>
    3 4 7 1 2 9 9<br>
    5<br>
    1<br>
    2<br>
    3<br>
    4<br>
    9<br>
- **Sample Output**<br>
    1<br>
    1<br>
    1<br>
    1<br>
    2<br>
