# **Chapter 3 - Assignment - 2 - Number of Swaps (Bonus)**

**[Task]**

**Problem**
- You have been given an array A of size N, you need to sort this array in non-decreasing order using bubble sort.
- However, you do not need to print the sorted array.
- You just need to print the number of swaps required to sort this array using bubble sort.

**Input Format**
- The first line contains a single integer N denoting the size of array *A*.
- The next line contains N space-separated integers denoting the elements of array *A*.

**Output Format**
- Print the required answer in a single line.

**Constraints**
- 1 ≤ N ≤ 1000
- 1 ≤ A[i] ≤ 1000

**Example 1**
- **Sample Input** <br>
    5<br>
    1 2 3 4 5<br>
- **Sample Output**<br>
    0<br>

**Example 2**
- **Sample Input** <br>
    5<br>
    1 3 2 5 4<br>
- **Sample Output**<br>
    2<br>
