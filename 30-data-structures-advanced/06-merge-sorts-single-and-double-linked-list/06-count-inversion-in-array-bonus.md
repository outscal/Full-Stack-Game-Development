# **Chapter 6 - Assignment - 2 - Count Inversion In Array (Bonus)**

**[Task]**

**Problem**
- You are given an array A of n integer numbers.
- Calculate the number of inversions in the array.
- Two elements A[i] and A[j] form an inversion if A[i] > A[j] and i < j.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the required answer.

**Constraints**
- 1 ≤ n ≤ 10^6
- 1 ≤ A[i] ≤ 10^9

**Example - 1**
- **Sample Input** <br>
    4<br>
    8 4 2 1<br>
- **Sample Output**<br>
    6<br>

**Example - 2**
- **Sample Input** <br>
    3<br>
    3 1 2<br>
- **Sample Output**<br>
    2<br>
