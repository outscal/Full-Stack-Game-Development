# **Chapter 6 - Assignment - 1 - XOR (Bonus)**

**[Task]**

**Problem**
- You are given an array A of n integer numbers.
- Calculate the number of pair of indices (i, j) such that 1≤ i, j ≤ n and A[i] xor A[j] = 0.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of array.

**Output Format**
- Output the required number of pairs.

**Constraints**
- 1 ≤ n ≤ 10^6
- 1 ≤ A[i] ≤ 10^9

**Example**
- **Sample Input** <br>
    5<br>
    1 3 1 4 3<br>
- **Sample Output**<br>
    2<br>
