**Problem - Permutations**
  - Given an array *nums* of distinct integers.
  - Return all the possible permutations.
  - You can return the answer in any order.
  
**Input Format**
  - First line: n denoting the number of array elements.
  - Second line: n space-separated integers, denoting elements of the array *nums*.

**Output Format**
  - Output each permutation in a new line.


**Constraints**
  - 1 ≤ n ≤ 10
  - -10 ≤ nums[i] ≤ 10


**Example**
  - **Sample Input** <br>
  3<br>
  1 2 3
  - **Sample Output**<br>
  1 2 3<br>
  1 3 2<br>
  2 1 3<br>
  2 3 1<br>
  3 1 2<br>
  3 2 1
