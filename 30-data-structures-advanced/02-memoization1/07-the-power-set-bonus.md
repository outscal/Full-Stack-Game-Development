**[Task]**

**Problem - Subsets II**
- Given an array *nums* of integers, that might contain duplicates
- Return all the unique ***subsets (the power set)***.
- You can return the answer in any order.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of the array *nums*.

**Output Format**
- Output each subset in a new line.

**Constraints**
- 1 ≤ n ≤ 10
- -10 ≤ nums[i] ≤ 10

**Example - 1**

- **Sample Input** <br>
    3<br>
    1 2 2
- **Sample Output**<br>
    1<br>
    2<br>
    1 2<br>
    2 2<br>
    1 2 2

**Example - 2**
- **Sample Input** <br>
    1<br>
    0
- **Sample Output**<br>
    0<br>
- **Note - Empty Subset is also included in the power set.**
