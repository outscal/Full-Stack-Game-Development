**Problem -**
- The **n-queens** puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other i.e. no two queens share the same row, column, or diagonal.
- Given an integer n, print the number of distinct solutions to the n-queens puzzle.

**Input Format**
- First line: one integer n.

**Output Format**
- Output the required answer.

**Constraints**
- 1 ≤ n ≤ 9

**Example - 1**
- **Sample Input** <br>
    4<br>
- **Sample Output**<br>
    2<br>
