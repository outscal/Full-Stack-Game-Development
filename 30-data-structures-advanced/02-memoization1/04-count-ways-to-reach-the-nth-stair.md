# **Chapter 2 - Assignment - 2 - Count ways to reach the n’th stair**

**[Task]** 

- There are n stairs, a person standing at the bottom(0th stair) wants to reach the top(nth stair). The person can climb either 1 stair or 2 stairs at a time. Write a program to count the number of ways, the person can reach the top.

**[Examples]**

**Input :** n = 1<br>
**Output :** 1<br>
**Explanation :** (0 → 1)


**Input :** n = 2<br>
**Output  :** 2<br>
**Explanation :**  (0 → 1 → 2) and (0 → 2) <br>


**Input :** n = 4<br>
**Output :** 5<br>
**Explanation** - (0 → 1 → 2 → 3 → 4) , (0 → 1 → 3 → 4), (0 → 2 → 3 → 4), (0 → 1 → 2 → 4) & 
                          (0 → 2 → 4)
- Note:- Solve using Recursion.
