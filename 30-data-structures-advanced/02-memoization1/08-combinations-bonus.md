# **Chapter 2 - Assignment - 6 - Combinations (Bonus)**

**[Task]**

**Problem -**
- Given two integers n and k.
- Print all possible combinations of k numbers out of the range [1, n].
- You can print the answer in any order.

**Input Format**
- First line: two integers n and k.

**Output Format**
- Output each combination in a new line.

**Constraints**
- 1 ≤ n ≤ 20
- 1 ≤ k ≤ n

**Example - 1**
- **Sample Input** <br>
    4 2<br>
- **Sample Output**<br>
    1 2<br>
    1 3<br>
    1 4<br>
    2 3<br>
    2 4<br>
    3 4<br>

**Example - 2**
- **Sample Input** <br>
    1 1<br>
- **Sample Output**<br>
    1<br>
