# **Chapter 12 - Assignment - 2 - Max Distance Between 2 Occurrences (Bonus)**

**[ Task ]**

**Problem**
- Given an array with repeated elements.
- Find the maximum distance between two occurrences of an element.

**Input Format**
- First line: Integer N denoting the size of the array.
- Next line: N space-separated integers denoting elements of the array.

**Output Format**
- Print the maximum distance.

**Constraints**
- 1 ≤ N ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9

**Example**
- **Sample Input** <br>
    7<br>
    3 4 7 1 2 9 4<br>
- **Sample Output**<br>
    5<br>
