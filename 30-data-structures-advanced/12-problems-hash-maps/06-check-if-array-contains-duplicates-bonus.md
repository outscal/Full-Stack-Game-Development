# **Chapter 12 - Assignment - 3 - Check if Array Contains Duplicates (Bonus)**

**[ Task ]**

**Problem**
- Given an unsorted array that may contain duplicates.
- Also given a number K which is smaller than the size of the array.
- Return true if the array contains duplicates within K distance.

**Input Format**
- First line: Integer N denoting the size of the array.
- Next line: N space-separated integers denoting elements of the array.
- Next line: Integer K

**Output Format**
- Return true of false accordingly.

**Constraints**
- 1 ≤ N ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9
- K ≤ N

**Example - 1**
- **Sample Input** <br>
    8<br>
    1 2 3 4 1 2 3 4<br>
    4<br>
- **Sample Output**<br>
    true<br>

**Example - 2**
- **Sample Input** <br>
    8<br>
    1 2 3 4 1 2 3 4<br>
    3<br>
- **Sample Output**<br>
    false<br>
