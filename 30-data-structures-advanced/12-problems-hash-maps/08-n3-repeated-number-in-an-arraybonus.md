# **Chapter 12 - Assignment - 5 - N/3 repeated number in an array(Bonus)**

**[ Task ]**

**Problem**
- Given an integer array of size n.
- Find all elements that appear more than ⌊n/3⌋ times.
        - ⌊n/3⌋ denotes floor (n/3)

**Input Format**
- First line: Integer n denoting the size of the array.
- Next line: n space-separated integers denoting elements of the array.

**Output Format**
- Print the required elements.

**Constraints**
- 1 ≤ n ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9

**Example - 1**
- **Sample Input** <br>
    3<br>
    3 2 3<br>
- **Sample Output**<br>
    3<br>

**Example - 2**
- **Sample Input** <br>
    1<br>
    1<br>
- **Sample Output**<br>
    1<br>
