# **Chapter 12 - Assignment - 4 - Common Elements Present In All Rows (Bonus)**

**[ Task ]**

**Problem**
- You are given a 2D matrix of size M*N.
- Find all the elements that are present in all rows

**Input Format**
- First line: Integers M,  N denoting the size of the 2D matrix.
- Next M lines: N space-separated integers denoting elements of the matrix.

**Output Format**
- Print required numbers.

**Constraints**
- 1 ≤ N, M ≤ 10^5
- 1 ≤ M*N ≤ 10^5
- 1 ≤ arr[i][j] ≤ 10^9

**Example - 1**
- **Sample Input** <br>
    4 4<br>
    1 2 3 4<br>
    2 3 4 5<br>
    3 4 5 6<br>
    4 5 6 7<br>
- **Sample Output**<br>
    4<br>
