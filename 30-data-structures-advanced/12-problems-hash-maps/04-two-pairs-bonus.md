# *Chapter 12 - Assignment - 1 - Two Pairs (Bonus)**

**[Task]**

**Problem**
- Given an array of distinct integers.
- Find if there are two pairs (a, b) and (c, d) such that a+b = c+d and a, b, c and d are distinct elements.

**Input Format**
- First line: Integer N denoting the size of the array.
- Next line: N space-separated integers denoting elements of the array.

**Output Format**
- Print **YES** if pair exists else **NO**.

**Constraints**
- 1 ≤ N ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9

**Example**
- **Sample Input**<br> 
    7<br>
    3 4 7 1 2 9 8<br>
- **Sample Output**<br>
    YES<br>
