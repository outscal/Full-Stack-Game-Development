# **Chapter 22 - Assignment - 1 - Minimum number of operation required to convert number x into y** (Bonus)

**[ Task ]**

**Problem**
- Given an initial number x and two operations which are given below:
    1. Multiply number by 2.
    2. Subtract 1 from the number.
- The task is to find out the minimum number of operations required to convert the number x into y using only the above two operations.
- We can apply these operations any number of times.

**Input Format**
- First line: x, the initial number
- Second line: y, the final number

**Output Format**
- A single integer denoting the minimum number of operations, if not possible then print -1.

**Constraints**
- 1 ≤ x, y ≤ 1000

**Example - 1**
- **Sample Input -** <br>
    2 <br>
    5 <br>
- **Sample Output -** <br>
    4 <br>

**Example - 2**
- **Sample Input -** <br>
    4<br>
    7<br>
- **Sample Output -** <br>
    2 <br>
