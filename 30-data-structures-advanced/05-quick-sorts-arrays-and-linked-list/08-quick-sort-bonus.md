# **Chapter 5 - Assignment - 3 - Quick Sort (Bonus)**

**[Task]**

**Problem**
- You are given an array A of n integer numbers.
- Sort the array in non-increasing order using quick sort algorithm.

**Input Format**
- First line: n denoting the number of array elements.
- Second line: n space-separated integers, denoting elements of the array.

**Output Format**
- Output the sorted array.

**Constraints**
- 1 ≤ n ≤ 10^6
- 1 ≤ A[i] ≤ 10^9

**Example - 1**
- **Sample Input** <br>
    4<br>
    8 4 2 1<br>
- **Sample Output**<br>
    8 4 2 1<br>

**Example - 2**
- **Sample Input** <br>
    3<br>
    3 1 2<br>
- **Sample Output**<br>
    3 2 1<br>
