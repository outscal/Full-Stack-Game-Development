# **Chapter 5 - Assignment - 1 - Prom Night (Bonus)**

**[Task]**

**Problem**
- The Hexagon University of India will be hosting its Prom Night tonight. There would be M boys and N girls at the prom tonight.
- Each boy wants a girl who is strictly shorter than him.
- A girl can dance with only one boy and vice-versa.
- Given the heights of all the boys and girls tell whether it is possible for all boys to get a girl.

**Input Format**
- The first line contains an integer T denoting the number of test cases.
- Each test case contains three lines.
    1. The first line contains M and N.
    2. The second line contains M integers each denoting the height of a boy.
    3. The third contains N integers each denoting the height of a girl.

**Output Format**
- Print YES if it is possible for each boy to get a girl else print NO.

**Constraints**
- 1 ≤ T ≤ 10
- 1 ≤ N, M ≤ 10000
- 1 ≤ height ≤ 500

**Example**
- **Sample Input** <br>
    1<br>
    4 5<br>
    2 5 6 8<br>
    3 8 5 1 7<br>
- **Sample Output**<br>
    Yes<br>
