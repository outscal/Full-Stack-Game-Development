# **Chapter 5 - Assignment - 2 - Find The Next (Bonus)**

**[Task]**

**Problem**
- You are given an array A of length N.
- For any given integer X, you need to find an integer Z strictly greater than X such that Z is not present in the array A. You need to minimize the value of Z.

**Input Format**
- First line: Two space-separated integers N and Q denoting the number of elements in array A and the number of queries respectively.
- Second line: N space-separated integers denoting the array elements.
- Next Q lines: Each line consists of an integer X.

**Output Format**
- Print Q lines, each line denoting the answer to the corresponding query.

**Constraints**
- 1 ≤ N, Q ≤ 10^5
- 1 ≤ A[i], X ≤ 10^9

**Example**
- **Sample Input** <br>
    5 2<br>
    2 7 5 9 15<br>
    3<br>
    6<br>
- **Sample Output**<br>
    4<br>
    8<br>
