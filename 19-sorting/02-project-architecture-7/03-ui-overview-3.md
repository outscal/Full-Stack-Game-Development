## UI CONTROLLERS:

- **SplashScreenUIController: **Shows the initial game screen when you first start the game.
- **MainMenuUIController: **Manages the main menu where you can choose different types of sorting algorithms.
- **GameplayUIController: **Controls the in-game interface, such as time complexity, visualizer, etc. You have already created it in Searching, here it is given in Setup.



## UI ELEMENTS:

- **ImageView:** Displays static images like backgrounds and icons in the game.
- **ButtonView:** A clickable ImageView that responds to mouse clicks, used for interactive buttons.
- **AnimatedImage:** Displays images with animations, such as fade transitions.
- **TextView:** Shows all on-screen text, with options to change font style and size for different UI elements.
