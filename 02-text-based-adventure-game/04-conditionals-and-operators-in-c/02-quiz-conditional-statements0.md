### Q1. What is the correct syntax for the if statement?

1. if { expression } {}
2. if expression {}
3. if ( expression ) {}
4. if [ expression ] ()
5. if ( expression ) []

- 3, 5
- 3, 4, 5
- 3 **(correct)**

### Q3. Given the code snippet:

int x = 10, y = 20;
if (x == 10) {
    if (y == 20) {
        cout << "x is 10 and y is 20" << endl;
    } else {
        cout << "x is 10 but y is not 20" << endl;
    }
} else {
    cout << "x is not 10" << endl;
}


What will be the output?
- x is 10 and y is 20 **(correct)**
- x is 10 but y is not 20
- x is not 10
- No output due to a compilation error
