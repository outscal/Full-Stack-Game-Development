# Creating Levels


---

## [tasks]

- Create 5 different Levels/Scenes.
- Add these scenes in the build settings.
- Implement `**LevelManager.cs**` script.
- Update the `**Player.cs**` script according to the `**LevelManager.cs**` script.



# UI in Mr. Blocks


---

## **Create Level UI:**
[tasks]

- Create the complete `**Canvas_Level**`.
- Create and Implement the `**LevelUI.cs**` script.
- Attach the `**LevelUI.cs**` script to the `**Canvas_Level**`.



## Create Main Menu UI:
[tasks]

- Create a new `**Main Menu**` scene and add it to the build settings.
- Create and Implement the `**MainMenuUI.cs**` script.
- Attach the `**MainMenuUI.cs**` script to the `**Canvas_MainMenu**`.





# Play the Sound


---

## [tasks]

- Create a new GameObject, name it "SoundManager."
- Add two `**AudioSources**` as child objects: "Bgm Source" and "Sound FX Source."
- On "Bgm Source," check the "Loop" box in the Inspector.
- Create and Implement the `**SoundManager.cs**` script.
- Attach the `**SoundManager.cs**` script to the SoundManager game object.
- Update the `**Player.cs**`, `**MainMenuUI.cs**`, and `**LevelUI.cs**` scripts according to the `**SoundManager.cs**`.
