## Download and Install Visual Studio Community


---


[Watch video](https://www.loom.com/embed/bc4fa6f20de145bb831aa3eef6a1fed4)





1. Visit the official Visual Studio Community download page: [https://visualstudio.microsoft.com/vs/community/](https://visualstudio.microsoft.com/vs/community/)
2. Download the Visual Studio Community installer for your OS.
3. Run the installer.
4. In the installer, select the "Game Development with Unity" workload.

This includes:

- Unity support for C# scripting
- Unity Debugger
- IntelliSense for Unity-specific classes and methods

1. Click "`Install`" and wait for the setup to complete.
