## What is UI?


---

UI is like the control panel of your game.

All the on-screen elements help you interact with the game and understand what's happening.

Think of it as the bridge between you and the game world.

For example, in a platformer game, UI elements might include:

- Lives remaining
- Score
- Timer

These help you know how you're doing without interrupting the gameplay.



## Different Types of UI in Games


---

Let's look at two types of UI you'll find in most games:

1. **Menus**: These help you navigate through the game options.
2. **HUD (Heads-Up Display)**: This shows important info while you're playing.



## UI That Changed Gaming


---

Some games have UI so good, they changed how other games are made. Here are a few examples:



![UI in Games](//outscal.github.io/Full-Stack-Game-Development/images/873bef11fa4f91ab.png)

**UI in Games**



1. **Halo (2001)**: Its HUD was one of the first to smoothly show health, shield, and ammo in a natural way for first-person shooters.
2. **Resident Evil (1996)**: This game's inventory system made item management a key part of the gameplay, adding to the tension of survival horror.
3. **Mass Effect (2007)**: This game's dialogue wheel made conversations in RPGs more intuitive and engaging.



## Why UI Matters


---

Good UI can make or break a game. It helps players:

- Understand what's happening in the game
- Make decisions quickly
- Stay immersed in the game world
