# Adding Physics to the Game


---

## [Tasks]

- Add `**Circle Collider 2D**` component to all three spikes.
- Add a `**Box Collider 2D**`** **and a** **`**Rigidbody 2D**`** **component to the `**Player**` game object.
- Assign "Player" and "Obstacles" tags to the Player and Spikes respectively.
- Implement the code for collision detection between Player and Spikes in the `**Player.cs**` script.
- Create a `**LevelFinishTrigger**`
- Implement the code for trigger detection in the `**Player.cs**` script.
- Update the code for movement in the `**Player.cs**` script.
