## How to Create a Prefab?


---

Creating a Prefab is actually quite simple. Here's how you do it:

1. Create a folder: `Assets/Prefabs/Player`
2. Select the `Player` Game Object in the `Hierarchy`.
3. Drag and Drop `Player` into the `Project` window in the new folder.
4. - This creates a Prefab out of Player.
5. You'll see Mr. Blocks turn blue in the** **`Hierarchy`, indicating that it's now a Prefab.
