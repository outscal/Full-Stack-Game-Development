## Creating Your First Unity Project


---

Here's how to set up your project:




[Watch video](https://www.loom.com/embed/9e5ca4ac19cf403bbc53dcaf419da4fc?sid=c02b7314-9364-43b3-afe9-02d6745a313c)





1. Open Unity Hub.
2. Click `New Project`.
3. Select the editor version as `2021.3.21f1 LTS`
4. Choose the `***2D (Built-In Render Pipeline)***` template.
5. Name your project *Mr. Blocks.*
6. Choose your save location.
7. Deselect “***Connect to Unity Cloud***” checkbox
8. Hit `Create Project`!



> 🎮📖 **Unity Dictionary**: 
> ***Unity Hub***: An application to manage Unity projects and installations. It allows users to organize projects, manage different versions of Unity, and create new projects.

## Adding the Assets


---

Good news! We've prepared some assets for you to play with. These will help you start and give you some cool stuff to experiment with.

Let's add them to your project:




[Watch video](https://www.loom.com/embed/825d7698d4be4c318d01fe09661c662d?sid=3d963ae9-5b7a-4f03-b653-c726c4b7a3e3)





1. Download the Unity Package: [Mr. Blocks Package](https://outscal-content.s3.ap-south-1.amazonaws.com/Mr_Blocks_Pacakge_87ce869f31.unitypackage)
2. In Unity, go to `Assets` > `Import Package` > `Custom Package`.
3. Select the downloaded package.
4. Click `Import` (make sure all assets are checked).
5. The assets appear in your `Project` Panel!



> 🎮📖** Unity Dictionary **
> ***Unity Package***: A collection of assets, scripts, or project settings bundled together to import into Unity projects. Packages can contain anything from simple assets to complete projects.
