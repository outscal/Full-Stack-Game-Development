> Create a **Bonus-Solution** branch and complete the following tasks, commit all these changes and submit the link of your branch for this assignment.



Your task is to craft an achievement system for this project, that enhances player engagement and reward their accomplishments



![](//outscal.github.io/Full-Stack-Game-Development/images/76534779e1e891c9.gif)



When a certain condition or event occurs, the corresponding achievement should be triggered and a UI Popup indicating that achievement has been unlocked must be shown for a few seconds.



You need to create the following achievements, but you can introduce your own unique ones as well:

- **KEYMASTER: **Collect all keys scattered throughout the haunted mansion.
- **SANITY SAVER: **Unlocked by finding and using all sanity-restoring items in the game.
- **TORMENTED SURVIVOR:** Escape the mansion with 90% or more insanity.
- **MASTER OF SHADOWS:** Spend more than 2 minutes in rooms with all lights-off.
