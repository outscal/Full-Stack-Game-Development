# **How to start building connections**



- Now it's time to start connecting with other people on LinkedIn.
- Don't be a noob and do the mistake of just pressing that blue connect button on your screen and thinking that you have made a meaningful connection with someone.
- You can start a conversation with that person before even you are connected with them by adding a message note with your connection request.
- Adding a message note allows you to personalize the connection request and make it more meaningful.
- It shows that you have taken the time to write a tailored message, rather than using the default automated request.
- Personalized connection requests tend to have a higher acceptance rate.
- When people see a genuine reason for connecting, they are more likely to accept the request than when they receive a generic one.
- A personalized message can help you stand out from other connection requests.
- When people receive numerous requests without any context, yours will be more memorable and leave a positive impression.
