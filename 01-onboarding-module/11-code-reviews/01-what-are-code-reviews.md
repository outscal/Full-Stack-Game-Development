Whenever you submit your code for review, one of our code reviewers will pick it up from the pool and review it **asynchronously within 24 hours**.



They will provide you detailed feedback on how to improve your code, and what is wrong with its current state.



If you really want to upskill as a programmer, frequent code review cycles are very necessary to keep a solid feedback loop.



The code reviewers will share a **video recording** of going through your code and explaining where it can be improved.


They will also give you a detailed TODO list to make improvements and a **PASS** or **FAIL** status. 



Your code review will be treated as strictly as any other coding assignment given by companies while hiring. 



If you FAIL the code review, you will need to go back and do the recommended changes and submit that code again for another review, until you are passed.
