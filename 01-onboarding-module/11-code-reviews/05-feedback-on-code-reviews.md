At Outscal we foster a culture of transparent feedback as it lets each of us grow.



It is necessary for you to providing feedback after receiving each code review because it helps us in ensuring that you are receiving quality reviews and are able to achieve your goals.



For each code review you receive, you will need to fill out a **feedback form** for the code reviewer.

You will need to share your experience for that particular code review in the feedback form.

If you feel like the review was not helpful for you or if there is any improvement that you would like to see, please share it in the feedback form.



It will allow us to address any issues regarding low quality code reviews in case they happen.
