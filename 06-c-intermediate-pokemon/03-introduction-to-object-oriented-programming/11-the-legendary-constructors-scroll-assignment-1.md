As you journey through the world of Pokémon, 

you stumble upon an ancient temple known as the "Shrine of Constructors." 



It's said that within this shrine lie hidden scrolls that hold the secrets of creating legendary Pokémon and Trainers. 



To unlock the full potential of your Pokémon and become a master Trainer, you must decipher and master these ancient scrolls!



Each scroll represents a unique type of constructor that can summon and shape powerful beings. 



Complete these challenges to gain control over the essence of creation!



### **Task 1: The Scroll of Origins – Default Constructor**



The first scroll teaches the art of summoning Pokémon and Trainers with default attributes.



- **Objective**: Create a default constructor for the `Player` class.



- - Inside the `Player` class, create a default constructor.



- - Initialize the name of the Player to `"Trainer"` and set the chosen Pokémon to a default Pokémon, Pikachu.



- **Objective**: Create a default constructor for the `Pokemon` class.



- - Inside the `Pokemon` class, create a default constructor.



- - Initialize the Pokémon’s name to `"Pikachu"`, type to `ELECTRIC`, and health to `10`.



### **Task 2: The Scroll of Customization – Parameterized Constructor**



The second scroll reveals the secret to customizing and personalizing your summons. 

With this power, 

you can create Pokémon and Trainers with unique strengths and identities.





- **Objective**: Create a parameterized constructor for the `Player` class.



- - Inside the `Player` class, create a parameterized constructor that takes `p_name` and `p_chosenPokemon` as arguments.



- - Initialize the Player's name and chosen Pokémon with these parameters.



- **Objective**: Create a parameterized constructor for the `Pokemon` class.



- - Inside the `Pokemon` class, create a parameterized constructor that takes `string p_name`, `PokemonType p_type`, and `int p_health` as arguments.



- - Initialize the Pokémon’s attributes with these parameters.



### **Task 3: The Scroll of Reflection – Copy Constructor**



The final scroll speaks of a mystical Pokémon that can mirror any creature it faces. 

This scroll grants the power to create perfect copies of your Pokémon and Trainers, 

just like the mythical Ditto!





- **Objective**: Create a copy constructor for the `Player` class.



- - Inside the `Player` class, create a copy constructor that takes a reference to another `Player` object.



- - Initialize the name and chosen Pokémon of the Player with the attributes of the referenced Player.



- **Objective**: Create a copy constructor for the `Pokemon` class.



- - Inside the `Pokemon` class, create a copy constructor that takes a reference to another `Pokemon` object.



- - Initialize the Pokémon’s attributes with the attributes of the referenced Pokémon.
