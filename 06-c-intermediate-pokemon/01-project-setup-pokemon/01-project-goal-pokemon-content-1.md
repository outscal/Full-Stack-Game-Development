# Project Overview


---

**What's ahead: **

You will create a text-based Pokémon game in C++ while applying the concept of OOPs to it. 

I know some of you guys are scared of this OOPs concept, but trust me, you will love this topic after completing this module and will know how to use it
to develop any game.



**How your game will look like:** 

- The user will be playing the game in the terminal
- Players will be able to fight wild Pokemons.
- Players will be able to capture new Pokémon during the gameplay
- An inventory system will be implemented to store all the Pokémon
- Player will fight in the gyms and Pokemon Leagues to compete and become the best Pokemon Trainer.
