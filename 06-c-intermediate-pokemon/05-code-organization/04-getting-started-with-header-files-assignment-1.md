Alright, let’s dive into organizing your code a bit better. 

Today, we’ll start by setting up a simple header file. 



Don’t worry,

we’ll take it one step at a time.




---



**Step 1: Create Your First Header File**

1. First, let’s create a new file called `header.hpp`.
2. Inside this file, just write a single closing curly bracket `}`. I know, this seems odd, but we’re doing this to understand how files are included in C++.



**Step 2: Include Your Header File in Main**

1. Now, go to your `main.cpp` file.
2. Instead of the usual closing curly bracket at the end of your `main()` function, type `#include "header.hpp"`.



Give the code a run and see how it works. 



This simple exercise shows how header files can be included, 

almost like copy-pasting code from one file to another.
