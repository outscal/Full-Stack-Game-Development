### Q1. What will be the output of the code?

#include <iostream>
int main() {
    char ch = 'A';
    int num = 5 + ch;
    std::cout << "num: " << num << std::endl;
    return 0;
}

### Q2. What will be the output of the code?

#include <iostream>

### Q3. What will be the output of the code?

#include <iostream>

void print(double x) {
    std::cout << "double: " << x << std::endl;
}

void print(int x) {
    std::cout << "int: " << x << std::endl;
}

int main() {
    float val = 3.5f;
    print(val);
    return 0;
}

- int: 3
- double: 3.5 **(correct)**
- Compilation error due to ambiguity
- double: 3.500000
