---



**Tasks 1: Building the Grass Struct with Misty**



1. - **Misty’s Challenge: Create the Grass Patch!**
  - - Misty hands you a task list. First, create a header file called `grass.hpp` in the same folder where your `main.cpp` is located.



1. - **Misty Explains Structs:**
  - - Misty says, “To represent the grass, define a struct called `Grass`. This struct will hold all the details about the Pokémon hiding in it.”



1. - **Pokémon List:**
  - - “Let’s add some wild Pokémon!” says Misty. Use a `vector` to store the wild Pokémon that can hide in the grass. This vector should contain `WildPokemon` type values. Don’t forget to include `<vector>` and `<string>` headers!



1. - **Encounter Rate:**
  - - “How often do you think we’ll encounter these Pokémon?” Misty asks. Add an `int` variable to set the encounter rate.



1. - **Environment Description:**
  - - “Where are we setting up this grass?” Misty continues. Add a `string` to describe the environment, like "Forest" or "Cave."



**Tasks 2: Setting Up the Grass Patches**



1. - **Create the Grass Instance:**
  - - Misty nods approvingly. “Now, let’s create an instance of the Grass struct called `forestGrass`.”



1. - **Fill the Grass with Wild Pokémon:**
  - - Misty gestures towards the grass. “I want to see a Zubat, Caterpie, and Pidgey hiding here, with an 80% encounter rate! Can you make that happen?”
