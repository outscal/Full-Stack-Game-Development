### Q2. For an Entity Component System (ECS) in game development, which structure is best for storing components of entities where components are frequently added and removed?
- Array
- Vector
- Linked List
- Both Array and Vector **(correct)**

### Q3. In a turn-based strategy game, each player's actions are queued for processing in a cyclical order. Which data structure is best suited for managing this player queue, allowing for dynamic entry and exit of players?
- Array
- Vector
- Linked List **(correct)**
- Priority Queue

<details><summary>Explanation</summary>

A Linked List, especially a circular linked list, is ideal for managing a queue where elements (players) can dynamically enter and exit, as it can easily add or remove nodes without reallocating or reorganizing the entire structure.

</details>

### Q4. Which data structure is ideal for storing player positions in a real-time multiplayer game to ensure quick updates and retrievals?
- Array
- Vector
- Linked List
- Both Array and Vector **(correct)**

### Q5. Which data structure is typically used to manage free blocks in a memory allocation system, allowing for flexible splitting and coalescing of blocks?
- Array
- Vector
- Linked List **(correct)**
- Binary Tree

<details><summary>Explanation</summary>

Linked Lists are commonly used in memory management systems to track free blocks because they can easily adjust to block splitting and merging by changing links without moving actual memory content.

</details>
