## Implement Circular Linked List (CLL) and its Operations


---

## [IN A REPLIT PROJECT]



**TODO: In this assignment, you will implement a Circular Linked List (CLL) in C++. You will create the CLL class, define its structure, and implement essential operations such as insertion, deletion, traversal, and find in the middle. **

**The aim is to understand how Circular Linked Lists (CLL) work and practice working with nodes in a circular arrangement.**



1. **Create the Circular Linked List Class:**

- - Define a `Node` structure with `data` and a pointer to the `next` node.
  - Here `data` is just an `integer` variable.
  - Define a `CircularLinkedList` class and add `Node* head_node` property.

1. **Create the following operations:**

`void insertAtEnd(int value); `
`void insertAtBeginning(int value); `
`void deleteFromEnd(); `
`void deleteFromBeginning(); `
`void traverse(); `
`void findTheMiddle();`
`void insertAtIndex(int index, int value); `
`void deleteFromIndex(int index); `
`void reverse();`

1. **Implement the following operations:**

- - `insertAtEnd(int value)`**:** Add a node at the end of the list.
  - `insertAtBeginning(int value)`**:** Add a node at the beginning of the list.
  - `deleteFromEnd()`**:** Remove a node from the end of the list.
  - `deleteFromBeginning()`**:** Remove a node from the beginning of the list.
  - `traverse()`**:** Print all the nodes in the list.
  - `findTheMiddle()`**:** Find the middle node and print its data.
  - `insertAtIndex(int index, int value)`**:** Add a node at a specified position in the list.
  - `deleteFromIndex(int index)`**:** Remove a node from a specified position in the list.
  - `reverse()`**:** Reverse the circular linked list.
