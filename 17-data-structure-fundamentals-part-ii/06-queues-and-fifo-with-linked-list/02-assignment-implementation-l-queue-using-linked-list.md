# Problem Statement


---



Implement a Queue Class using a linked list which should have the following functions -

1. push(T x) - insert x into queue
2. pop() - remove the front element
3. front() - returns the front element
4. size() - returns the size of queue i.e. number of elements in queue
5. empty() - return true if queue is empty, false otherwise.
