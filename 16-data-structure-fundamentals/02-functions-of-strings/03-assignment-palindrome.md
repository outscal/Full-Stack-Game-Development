# Problem Statement



You have give a string ‘str’.

Check if the given string is Palindrome or not.

and if string is palindrome , **print “Yes”** and if not, **print “No”**.



**For Example**

```cpp
    Input : str = “aba”             
    `Output  = “Yes”.
```

```plain
   Input : str = “abcde”             
   `Output  = “No”`
```



**Note -**

1. A string is said to be palindrome if the reverse of the string is the same as the string.
2. String consists of only lowercase letters.
