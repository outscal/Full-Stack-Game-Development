# Problem Statement


---


Print the count of each unique character present in the input string ‘str.



**For Example**

         Input : str = “ABBAC”.  

         Output :  A - 2 

                        B - 2

                        C - 1



         Input : str = “abcAB”

         Output:  A - 1

                       B - 1

                       a - 1

                       b - 1 

                       c - 1



**Note -**

1. String can contain both lowercase and uppercase letters.
2. Try to use ASCII approach.
