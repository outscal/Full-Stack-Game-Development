# PROBLEM STATEMENT:


---

You have given a string `s`.

The **score** of a string is defined as the sum of the absolute difference between the **ASCII** values of adjacent characters.

Return the **score** of **`s`.



Example 1:

**Input:** s = "hello” 

**Output:** 13



**Explanation:** 

- The **ASCII** values of the characters in `s` are: `'h' = 104`, `'e' = 101`, `'l' = 108`, `'o' = 111`.So, the score of `s` would be :Now , add statements (1 + 2 + 3 + 4) =`|104 - 101| + |101 - 108| + |108 - 108| + |108 - 111| = 3 + 7 + 0 + 3 = 13`..
- 1. |h-e| = `|104 - 101|`
  2. |e-l| = `|101 - 108|`
  3. |l - l| = `|108 - 108|`
  4. |l-o| = `|108 - 111|`



Example 2:

**Input:** s = "zaz” 

**Output:** 50



**Explanation:** 

- The **ASCII** values of the characters in `s` are: `'z' = 122`, `'a' = 97`.So, the score of `s` would beNow , add statements (1 + 2 ) = `|122 - 97| + |97 - 122| = 25 + 25 = 50`.
- 1. |z-a| = `|122 - 97|`
  2. |z-a| = `|97 - 122|`



# Constraints:

- `2 <= s.length <= 100`
- `s` consists only of lowercase English letters.
