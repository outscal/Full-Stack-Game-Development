# Full-Stack Game Development

An open-source, structured learning path for game development — covering C++, C#, Unity, data structures, and design patterns through hands-on projects.

## About

This repository contains a comprehensive collection of modules that take you from programming fundamentals to advanced game development concepts. Each module is project-driven, so you learn by building real games and systems rather than reading theory alone.

Whether you're a beginner picking up C++ for the first time or an experienced developer looking to learn game design patterns, you'll find something useful here.
