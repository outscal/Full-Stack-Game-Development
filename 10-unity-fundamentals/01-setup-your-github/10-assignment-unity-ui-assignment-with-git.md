# **Assignment 6 - Unity UI Assignment with Git**

### [Task] Task - Create a Master scene - with 3 buttons (showing 1, 2, 3 as text on the button).

### [NOTE]

Make sure to create a respected branch as mentioned in submission instructions before starting the work

- Clicking button 1 - should load scene 1 with text 1.
- Clicking button 2 - should load scene 2 with text 2.
- Clicking button 3 - should load scene 3 with text 3.
- Each scene (1/2/3) has a back button in the top left corner - on clicking - go back to the Master scene.

### [Submission]

- Create branch named as Unity_UI_Basics
- Commit and push all your work as stated in Task.
- Create Pull requesting from your newly created branch and Turn In this assignment.
