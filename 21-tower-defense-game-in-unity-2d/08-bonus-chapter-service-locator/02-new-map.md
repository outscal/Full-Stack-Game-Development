# Design Your Own Map



> Create a **Bonus-Setup** branch and complete the following tasks, commit all these changes and submit the link of your branch for this assignment.



Congratulations on progressing through the Angry Monkeys tower defense course! As an additional challenge, we invite you to create a brand-new map for the game, offering a unique path for the balloons to follow.



![](https://tenor.com/view/swellview-map-police-officer-direction-henry-danger-gif-12642994.gif)



**Tasks:**

- Ensure that the map becomes accessible in the map selection menu only after the player successfully defends the first map.
- Implement a system where the second map becomes clickable and unlocked only after the player wins or defends the first map.
- Provide visual or textual cues indicating the locked status of the second map until the prerequisite conditions are met.
