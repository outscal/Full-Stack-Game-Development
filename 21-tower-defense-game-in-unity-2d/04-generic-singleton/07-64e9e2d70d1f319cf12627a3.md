# [Quiz] Disadvantages of Singleton

Disadvantages of Singleton


## Questions


### Q1. Which of the following is a potential disadvantage of the Singleton design pattern?
- It allows for easy extension of functionality without modifying existing code.
- It hides dependencies and can lead to unclear codebase structures. **(correct)**
- It simplifies unit testing and makes code more maintainable.
- It eliminates the need for global access to a single instance of a class.

### Q2. Why might the Singleton pattern be considered less favorable in terms of unit testing?
- It provides clear separation of concerns, making unit testing easier.
- It encapsulates complex logic, reducing the need for unit testing.
- It introduces global state, making unit tests dependent on specific instances. **(correct)**
- It enhances code modularity, making unit testing unnecessary.
